# Autonomous Car - Multi-Camera Obstacle Avoidance

Free-space navigation vehicle using Raspberry Pi 4B + Arduino Uno.

## Hardware
- Raspberry Pi 4B 4GB
- Arduino Uno R3 + L293D Motor Shield
- 4x DC Motor
- 2x Logitech C270 Camera (front + right)

## Software
- Python 3 + OpenCV 4.10
- Canny + Contour (static obstacle detection)
- HOG + SVM (person detection)
- pyserial (Pi to Arduino communication)
